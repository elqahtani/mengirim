# Images

Letakkan asset gambar di folder ini. File berikut di-referensikan oleh landing page:

- `dashboard-mockup.webp` — screenshot dashboard utama (rekomendasi: 1920×1080, max ~200KB).
- `og-image.webp` — gambar Open Graph untuk social share (rekomendasi: 1200×630).
- `logo.webp` — logo untuk structured data (rekomendasi: 512×512, transparan).

Landing page sudah punya fallback graceful kalau file ini belum ada, jadi aman untuk dev.
